## Image Sources:
- Use high-quality royalty-free images from:
  - Unsplash (unsplash.com)
  - Pexels (pexels.com)
  - Resort websites (with permission)
  - Original photography

## Optimization:
- Compress images using TinyPNG or similar
- Target: <200KB per image
- Format: JPG for photos
- Dimensions: 1400x600px (16:9 aspect ratio for hero)
- Card images: 800x450px recommended

## Image Guidelines:
- Show pickleball courts when possible
- Include destination context (beaches, mountains, etc.)
- High quality, well-lit photos
- Avoid overly-busy compositions
- Consider focal point for text overlay on hero images
